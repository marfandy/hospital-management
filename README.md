# hospital-management

## Quick Start

Prerequisites

- git
- python >= 3.8

This presume that you have installed apps above.

1. Clone this project

    ```bash
    git@github.com:marfandy/hospital-management.git
    ```

2. Create `.env` and set your environment variables

    ```bash
   python -m venv venv
    ```

3. Install packages

    ```bash
    pip install -r requirements.txt
    ```

4. Start hacking!

    ```bash
    python index.py
    ```
